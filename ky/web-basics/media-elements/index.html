Медиа элементтери

GIF кош

GIF Graphic Interchange Format дегенди билдирет, бирок сиз билишиңиз керек болгон нерсе - бул сүрөт файлынын башка түрү. Бир гана айырмасы файлда көптөгөн сүрөттөр бар. Браузерде көрсөтүлгөндө, сүрөттөр тасманын кадрларына окшош ырааттуулукта ойнотулат. Аны оодарганыңызда сүрөт жандангандай көрүнгөн флип-китептердин бири сыяктуу элестетиңиз.

Видео кош

Тексттен жана сүрөттөрдөн тышкары, биз веб-сайттарга кошкон эң көп нерселердин бири бул видеолор. Видео файлдары абдан чоң, андыктан YouTube сыяктуу видеону жүктөө жана көрсөтүү үчүн кызматты колдонгонуңуз жакшы. YouTube'га жүктөлгөндөн кийин, аны кантип веб-сайтыбызда ойнота алабыз? Биз iframe деп аталган атайын тегди колдонобуз.

Iframes биздин сайттын ичинде башка веб-сайттардан HTMLди көрсөтөт. Бул терезе сыяктуу, анда сиз учурдагы веб-сайттын ичинен интернетте сыртта эмне болуп жатканын көрө аласыз. Бул процесс embedding деп аталат.
