**zip**

zip is a command that compresses a set of files.
Zip [name of zip file] [filenames to be zipped]

**tar**

tar is a command that creates and extracts archived files.

| flags | Uses |
| :--- | :--- |
| -c | creates archive |
| -x | extracts archive |
| -f | creates archive with given filename |

tar [filenames]
