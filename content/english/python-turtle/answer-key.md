### Activity 4
```python
def draw_hexagon():
  for i in range(6):
    turtle.forward(50)
    turtle.left(60)

draw_hexagon()
```

### Activity 5
```python
def draw_honeycomb():
  for i in range(6):
    draw_hexagon()
    turtle.forward(50)
    turtle.right(60)

draw_honeycomb()
```

### Activity 6

### Activity 7
```python
def figure_8():
  turtle.circle(50)
  turtle.circle(-50)

figure_8()
```

### Activity 8
```python
def mandala_flower():
  for i in range(35):
    figure_8()
    turtle.right(10)

mandala_flower()
```
