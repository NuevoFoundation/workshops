The components we will use for the project are listed below. Make sure you have all of these before starting: 

1. 2 AA batteries
2. Motor
3. 2 Switches: 
	1. Momentary/Limit (single-pole-double-throw “SPDT”) switch 
	2. Toggle switch (double-pole-double-throw “DPDT”)
4. Acrylic/plastic pieces to make the box.
5. Wire (needs further details)
