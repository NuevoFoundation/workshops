[EarSketch](https://en.wikipedia.org/wiki/EarSketch) é um ambiente de programação educacional gratuito desenvolvido no Georgia Institute of Technology. Estaremos compondo músicas usando o EarSketch hoje.

1. Para fazer música, clique no botão abaixo para abrir Earsketch em uma nova aba.

<a class="my-2 mx-4 btn btn-info" href="https://earsketch.gatech.edu/earsketch2/" target="_blank">Iniciar Earsketch</a>

2. Agora, vamos começar. Clique no ícone *Scripts* na barra de menu à esquerda e
selecione *Criar um novo script*. Quando solicitado a inserir o nome do script, digite *myjam* e clique no botão *Criar*.

![alt text height="600px" width="60%"](../gif/createanewscript.gif "Novo script de demonstração")

3. Não exclua nenhum código pré-existente! Vamos inserir nosso código entre eles.

## Recomendado para professores - visão geral do EarSketch

Por favor, dê uma olhada na visão geral do EarSketch antes de começar a aula. É importante saber como navegar para frente e para trás na IU.

A `barra de navegação` no lado esquerdo da tela é sua base para navegar por tudo o que o EarSketch tem a oferecer.

1. O primeiro item da barra de navegação é a coleção `Sounds`. É o local central para todos os sons pré-carregados que o earsketch tem a oferecer. No navegador de som, você pode filtrar quais sons aparecem com base nos artistas, gênero ou instrumentos.
    ![](../img/screenshot-navigation-sound-browser.png)

2. O próximo ícone na barra de navegação é o painel `Scripts`. É aqui que escreveremos nosso código.

     ![](../img/screenshot-navigation-script.png)

3. O próximo (terceiro de cima) ícone é a seção `Compartilhar`. Você pode usar isso para salvar seu código, baixar ou compartilhar com seus amigos. Vamos pular esta seção para este exercício.

4. Os últimos dois ícones na barra de navegação são os guias `API Browser` e `Curriculum`.

     ![](../img/screenshot-navigation-api-curriculum.png)

Usaremos o `navegador API` para fazer música. Uma *Application Programming Interface* ou *API* é um conjunto de códigos que está pronto para você usar imediatamente! Você pode notar dois ícones à direita de cada API. O primeiro ícone fornece uma descrição da API e um exemplo de como usar a API. O segundo ícone copia e cola a API no editor de código, então você nem precisa digitar! Nós
usaremos alguns deles nas próximas seções.

O último ícone na imagem acima mostra o guia `Currículo`. Se você quiser saber mais sobre o EarSketch, pode explorar a seção que aparece no lado direito da janela quando você clica nela. Vamos pular esta seção para este exercício.
