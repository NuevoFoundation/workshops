## Conheça o Visual Studio Code

O Visual Studio Code é um editor de código, com uma interface de usuário e layout semelhantes a muitos outros editores de código. No lado esquerdo da interface há uma barra lateral que você usará para acessar os arquivos e pastas do seu projeto de código. No lado direito está a área do editor, que exibe o conteúdo dos arquivos da aplicação.

### Entenda as principais áreas da interface do usuário

O diagrama a seguir mostra as seis principais seções incluídas na interface do usuário do Visual Studio Code.

<img src="../media/visual-studio-code-user-interface-labels.png" alt="Captura de tela mostrando 5 áreas da interface do usuário do VS Code" />

A - A **Barra de Atividades** é a barra vertical estreita localizada no lado esquerdo da janela. A Barra de Atividades permite alternar entre visualizações (como Explorador ou Extensões) e fornece outros indicadores específicos de contexto.

B - A **Barra Lateral** contém visualizações que fornecem ferramentas e recursos. As visualizações da Barra Lateral (como EXPLORADOR) são muito úteis quando você está trabalhando em um projeto de código.

C - O **Editor** é a área usada para editar seus arquivos. Você pode abrir quantos editores quiser, lado a lado, vertical e horizontalmente.

D - A área do **Painel** é usada para exibir diferentes painéis abaixo da região do editor para saída ou informações de depuração, erros e avisos, ou um terminal integrado.

E - A **Barra de Status** é a barra horizontal na parte inferior da janela que exibe informações sobre o projeto aberto e os arquivos que você edita.

F - O **Menu Principal** é a interface de menu na parte superior da janela do aplicativo. As opções do menu permitem salvar, editar e executar seu código (e muito mais).
