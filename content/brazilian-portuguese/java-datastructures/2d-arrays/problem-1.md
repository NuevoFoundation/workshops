```js javascript

                    { "Hamburger", "Pasta", "Steak", "Tacos" },
                    { "Cola", "Sprite", "Pepsi", "Fanta" },
                    { "Pudding", "Ice Cream", "Mochi", "Cake" }
                  };
```
