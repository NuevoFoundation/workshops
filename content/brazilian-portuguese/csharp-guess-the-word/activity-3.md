## Atividade 3

Esta atividade é dividida em 2 partes e vai testar seu conhecimento sobre variáveis. Antes de começar, procure por `HINT (ACTIVITY 3)` no código, onde declaramos uma variável chamada `lives` que guarda o número de vidas do jogador, e usamos essa variável para mostrar quantas vidas o jogador tem no início do jogo. Isso deve te dar uma ideia de como completar as duas partes a seguir.

### Atividade 3.1

No início do jogo, devemos dizer ao jogador quantas letras a palavra para adivinhar tem. Por exemplo, se a palavra é `"Hello"`, devemos dizer: `The word has 5 letters`. Se a palavra é `"Foundation"`, devemos dizer: `The word has 10 letters`. Já armazenamos o número de letras na variável `numberOfLetters`, então use-a para completar esta atividade.

Para verificar se completou esta atividade corretamente, aperte **run**, e veja se aparece `The word has 5 letters` (ou algum outro número diferente de 5) no console.

### Atividade 3.2

Existem mais duas informações que precisamos que o computador guarde:

- O número de letras que o jogador adivinhou corretamente até agora.
- Se o jogador ganhou ou não o jogo.

Vamos declarar duas variáveis para representar essas informações. Aperte **run** para garantir que não aparecem erros no console. Lembre-se que variáveis não produzem saída no console, então, nesta parte, apenas verificamos se não há erros.
