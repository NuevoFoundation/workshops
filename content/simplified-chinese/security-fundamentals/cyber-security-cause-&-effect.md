实践中，网络安全远不止是不在网上上当受骗。那么，用户在线上经常忽视的常见风险是什么，而这些又是如何被黑客利用的？ 

1. 未启用双重身份验证登录账户（社交媒体、银行、游戏等）  

这使得黑客更容易破解您的密码并在您不知情的情况下登录。

{{% notice tip %}}

### 启用双重身份验证 (2FA)
双重身份验证（通常缩写为 2FA）是指在登录时需要两种形式（或因素）来证明您的身份。这非常有帮助，因为即使有人偷走或猜到了您的密码，他们仍然无法登录到您的账户。

双重身份验证需要在潜在的安全漏洞发生**之前**启用。大多数账户在您首次创建账户时会要求您启用此功能，如果您当时没有设置，通常可以在账户设置页面中进行设置。系统会提示您提供一个电话号码或邮箱地址，用于进行身份验证。在提供第二种联系方式后，您将收到一条确认消息，以确保您可以接收到这些验证信息。

一旦启用了双重身份验证，每次登录尝试时都会向您提供的邮箱或电话号码发送消息，以确认登录者确实是您本人。确保登录时手机在身边！

{{% /notice %}}

2. 密码复用（在多个网站上使用相同的密码）

一旦您的密码被发现，黑客可能会访问您其他在线账户甚至所有账户。

{{% notice tip %}}

### 使用密码管理器

密码管理器是存储密码的安全工具，帮助您防止忘记密码。使用密码管理器有以下几个好处：

1. 您无需记住多个用户名和密码，只需记住用于解锁密码管理器的密码即可查看所有密码。
2. 密码管理器可以让您设置复杂的、难以被破解的密码，而不必担心记不住，因为您可以直接复制粘贴这些密码。
3. 大多数密码管理器可以为您推荐强密码，确保您不会使用弱密码。

有许多密码管理器可供选择，其中一些甚至内置于网页浏览器中！

{{% /notice %}}
