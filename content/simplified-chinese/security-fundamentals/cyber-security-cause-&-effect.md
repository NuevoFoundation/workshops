网络安全的实践远不止是避免在网上被骗。那么，用户常常忽视的网络风险是什么？而这些又是黑客利用的关键？

1. 未启用双因素认证来登录账户（社交媒体、银行、游戏等）

这会让黑客更容易破解你的密码，并在你毫不知情的情况下登录你的账户。

{{% notice tip %}}

### 启用双因素认证 (2FA)
双因素认证（通常简称为2FA）是在登录时需要两种形式（或因素）来证明您的身份的做法。这是一种有用的安全措施，因为即使别人窃取了或猜出了你的密码，他们仍然无法登录到你的账户。

2FA需要在潜在人为入侵发生**之前**启用。大多数账户在您首次创建账户时会提示您启用，如果您当时没有设置，可以在账户设置页面中找到相关选项。您需要提供一个电话号码或电子邮箱地址以确认您的身份。提供第二联系方式后，您会收到一条确认信息，以确保您能够收到验证信息。

启用2FA后，登录尝试会向您提供的电子邮箱或电话号码发送消息，以确认登录的人确实是您本人。请在登录时随身携带您的手机！

{{% /notice %}}

2. 密码重复使用（在多个网站使用相同的密码）

一旦黑客发现了您的密码，他们可能会访问您的更多或全部在线账户。

{{% notice tip %}}

### 使用密码管理器

密码管理器是一个安全存储密码的地方，这样您就不用担心忘记它们。使用密码管理器有以下几个好处：

1. 您不用记住许多用户名和密码，只需要记住一个密码来解锁密码管理器，查看所有您的密码。
2. 您可以设置复杂的密码，不用担心记住它们，因为可以直接复制粘贴密码。
3. 大多数密码管理器可以为您推荐强密码，确保您不会使用弱密码。

市场上有很多密码管理器可供选择，有些甚至内置在网络浏览器中！

{{% /notice %}}
