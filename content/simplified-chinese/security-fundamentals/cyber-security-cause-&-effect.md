实践中的网络安全远不止是不在网上被骗那么简单。那么，究竟有哪些常见风险是用户在线时忽视的，而这些风险又是黑客经常利用的？

1. 未启用双因素身份验证以登录账户（社交媒体、银行、游戏等）。

这会让黑客更容易破解您的密码并在您不知情的情况下登录您的账户。

{{% notice tip %}}

### 启用双因素身份验证 (2FA)
双因素身份验证（通常缩写为2FA）是指在登录时需要两种形式（或因素）来证明您的身份。这种方法非常有用，因为即使有人窃取或猜中了您的密码，他们仍然无法登录您的账户。

2FA需要在潜在的违规事件发生**之前**启用。大多数账户在您首次创建时会要求您设置2FA，但如果当时未设置，您仍然可以在账户设置页面中找到相关选项。您需要提供一个电话号码或电子邮件地址来确认您的身份。一旦您提供了第二种联络方式，系统会发送一条确认信息，以确保您能够接收到这些身份验证信息。

启用2FA后，每次登录尝试都会向您提供的电子邮件或手机号码发送消息，以确认登录者确实是您本人。在登录时，请确保手机在身边！

{{% /notice %}}

2. 重复使用密码（在多个网站上使用相同的密码）。

一旦您的密码被发现，黑客很可能能访问您所有的账户。

{{% notice tip %}}

### 使用密码管理器

密码管理器是一个安全存储您密码的地方，这样您不会忘记密码。使用密码管理器有几个好处：

1. 您不需要记住多个用户名和密码，只需记住解锁密码管理器的密码，便可以查看所有密码。
2. 您可以创建复杂且难以破解的密码，而无需担心记不住，因为可以直接复制粘贴密码。
3. 大多数密码管理器可以推荐强密码，以确保您不会使用弱密码。

市面上有许多类型的密码管理器，有些甚至内置于浏览器中！

{{% /notice %}}
