## Introducción
En este taller, aprenderás a crear tu propio chatbot. Al finalizar el taller, tendrás un chatbot que podrá mantener una conversación simple con el usuario. Aprenderás a utilizar AIML, un lenguaje para crear chatbots. También aprenderás a usar Pandorabots, una plataforma para alojar chatbots.

Para completar el taller, necesitarás una dirección de correo electrónico para registrarte en una cuenta de Pandorabots.

## ¿Qué es un chatbot?
Un chatbot es un programa informático que simula una conversación humana para responder preguntas. 
<img src="./img/surprised.png" alt="" width="20%" style="" />

## Ejemplos de chatbots
Es probable que ya hayas interactuado con un chatbot en tu vida cotidiana. 

Es posible que uses un asistente virtual para establecer un recordatorio en tu teléfono o para reproducir una canción en tu altavoz inteligente.

## Historia de los chatbots
Los chatbots existen desde hace mucho tiempo. El primer chatbot fue creado en 1966 y se llamó ELIZA. En 1995, se creó un nuevo chatbot llamado ALICE. ALICE fue escrito utilizando AIML, que es el mismo lenguaje que aprenderemos en este taller.
