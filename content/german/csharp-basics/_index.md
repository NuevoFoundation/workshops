## Übersicht

Willkommen zum Kurs **C#: Grundlagen**, in dem Sie eine weitverbreitete, hochentwickelte Programmiersprache kennenlernen, die Computeranweisungen erstellt, um spannende Programme zu entwickeln!

Die Programmiersprache C# wurde 2000 von Anders Hejlsberg bei Microsoft entworfen. Zum Zeitpunkt der Gestaltung dieses Kurses ist die aktuellste stabile Version von C# die Version 11.0, veröffentlicht im Jahr 2022.

{{% notice info %}}
Eine hochentwickelte Programmiersprache wird als hochentwickelt bezeichnet, weil sie einige Schritte vom ursprünglichen Code entfernt ist, der auf der Central Processing Unit (CPU) eines Computers ausgeführt wird. Hochentwickelter Code ist für Menschen gedacht und viel leichter zu verstehen als Code auf niedrigerer Ebene. Der hochentwickelte Code wird später in eine Niedrigsprache übersetzt, die nur von einer bestimmten CPU wahrgenommen und ausgeführt werden kann. Damit der Code funktioniert, muss jede einzelne Zeile Code, die in einer hochentwickelten Sprache geschrieben ist, in Maschinensprache übersetzt werden, bevor sie vom Computer implementiert wird.
{{% /notice %}}

C# gehört zu den beliebtesten Programmiersprachen der Welt, zusammen mit Java, JavaScript, Python und anderen. C# kann auf jedem Computer laufen, einschließlich Windows, Mac und allen Linux-Betriebssystemen.
