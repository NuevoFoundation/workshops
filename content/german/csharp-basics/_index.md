## Übersicht

Willkommen zum Kurs **C#: Grundlagen**, in dem Sie eine weitverbreitete, höhere Programmiersprache lernen, die Computeranweisungen übermittelt, um spannende Programme zu erstellen!

Die Programmiersprache C# wurde 2000 von Anders Hejlsberg bei Microsoft entworfen. Zum Zeitpunkt der Erstellung dieses Kurses war die neueste stabile Version der Sprache C# 11.0, veröffentlicht im Jahr 2022.

{{% notice info %}}
Eine höhere Programmiersprache wird so genannt, weil sie einige Schritte vom ursprünglichen Code entfernt ist, der auf der Zentraleinheit (CPU) eines Computers ausgeführt wird. Höheres Niveau-Code ist für Menschen gedacht und viel leichter zu verstehen als Code auf niedrigerem Niveau. Der höhere Niveau-Code wird später in eine niedrigere Sprache übersetzt, die nur von einer bestimmten CPU wahrgenommen und ausgeführt werden kann. Damit der Code funktioniert, muss jede einzelne Zeile Code, die in einer höheren Programmiersprache geschrieben wurde, in Maschinensprache übersetzt werden, bevor sie vom Computer ausgeführt wird.
{{% /notice %}}

C# ist eine der beliebtesten Programmiersprachen der Welt, zusammen mit Java, JavaScript, Python und anderen. C# kann auf jedem Computer laufen, einschließlich Windows, Mac und beliebigen Linux-Betriebssystemen.
