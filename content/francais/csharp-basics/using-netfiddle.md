### Utiliser .NET Fiddle
Aujourd'hui, vous allez utiliser .NET Fiddle pour apprendre à coder en C#.

Avant de commencer, comprenons d'abord comment fonctionne la fenêtre. Sur la gauche, vous avez une liste d'options. Celles-ci ont toutes été configurées pour vous, donc vous n'avez pas besoin de vous en préoccuper. Le centre de l'écran contient du code, ou des instructions pour l'ordinateur. En appuyant sur le bouton **run** (le bouton avec le triangle au milieu en haut du cadre), vous demandez à l'ordinateur d'exécuter les instructions contenues dans le code. Les résultats de l'exécution du code s'afficheront en bas de l'écran.

<img src="../images/donetfiddle-overview.png" height="300" alt="Disposition de la fenêtre .NET Fiddle" />

Passez un peu de temps à découvrir l'outil ci-dessous.

<iframe width="100%" height="475" src="https://dotnetfiddle.net/Widget/ccWNBp" frameborder="0"></iframe>

### Quelques conseils sur C#

Quelques conseils importants :
1. Chaque ligne de code se termine par un point-virgule `;`
2. Il y aura du code dans les exemples que vous ne comprendrez pas. Ce n'est pas grave, ignorez-le pour l’instant et concentrez-vous sur la tâche en cours.

## Sauvegarder votre travail

Si vous souhaitez sauvegarder votre travail tout au long de ces exercices, vous devrez vous inscrire pour un compte .NET Fiddle, si vous n’en avez pas encore.

Cliquez sur le bouton ci-dessous pour vous inscrire à un compte .NET Fiddle - vous aurez besoin d'un compte email pour vous inscrire. Si vous avez déjà un compte .NET Fiddle, vous pouvez vous connecter à votre compte en utilisant le bouton ci-dessous.

<a class="my-2 mx-4 btn btn-info" href="https://dotnetfiddle.net/SignUp" target="_blank">S'inscrire à .NET Fiddle</a>

<img src="../images/dotnetfiddle-signup.png" height="300" alt="Formulaire d'inscription .NET Fiddle" />

Ensuite, vous pouvez maintenant *forker* le Fiddle, ce qui signifie simplement faire une copie du programme, afin que nous puissions effectuer et enregistrer nos propres modifications. Cliquez sur le bouton "Fork" en haut.

Après avoir forké le Fiddle, vous verrez une fenêtre semblable à celle ci-dessous :

<img src="../images/dotnetfiddle-window.png" height="300" alt="Fenêtre .NET Fiddle" />
