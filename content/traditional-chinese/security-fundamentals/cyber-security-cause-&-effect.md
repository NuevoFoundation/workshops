實踐網路安全不僅僅是避免在線上受騙。究竟有哪些常見的風險是用戶經常忽略，而駭客卻能加以利用呢？

1. 未啟用雙重驗證來登入帳戶（例如社交媒體、銀行、遊戲等）

這會讓駭客更容易破解您的密碼並在不知情的情況下登入您的帳戶。

{{% notice tip %}}

### 啟用雙重驗證 (2FA)
雙重驗證（常簡稱為 2FA），是指在登入時需要提供兩種形式（或因素）來證明您的身份。這很有幫助，因為即使有人竊取或猜測了您的密碼，他們仍然無法登入您的帳戶。

必須 **在** 潛在的安全漏洞發生之前啟用 2FA。大多數帳戶在您第一次創建帳戶時就會提示您啟用，但如果當時未設置，您仍然可以在帳戶設定頁面找到相應選項。系統會提示您提供一個電話號碼或電子郵件地址，以用於身份驗證。提供第二種聯繫方式後，您會收到確認訊息，以確保您可以接收到這些驗證訊息。

啟用 2FA 後，每次嘗試登入時，都會將訊息發送至您提供的電子郵件或手機號碼，以確認登入者是否真的是您本人。在登入時務必保持手機在手邊！

{{% /notice %}}

2. 重複使用密碼（在多個網站上使用相同的密碼）

一旦您的密碼被發現，駭客很可能可以進一步存取您線上更多的帳戶。

{{% notice tip %}}

### 使用密碼管理器

密碼管理器是一個安全的地方，可以存放您的密碼以防忘記。使用密碼管理器有幾個好處：

1. 您不需要記住多組使用者名稱和密碼，只要記住用來打開密碼管理器的密碼，就能查看所有的密碼。
2. 您可以設置複雜的密碼，不用擔心記不住，因為您可以直接複製並貼上這些密碼。
3. 大多數密碼管理器可以為您推薦強密碼，確保您不會使用弱密碼。

目前市面上有許多密碼管理器可以使用，有些甚至已內建在網頁瀏覽器中！

{{% /notice %}}
