實踐網路安全不僅僅是避免在網路上受騙。究竟有哪些常見風險是使用者在網路上忽略的，而駭客利用了這些漏洞呢？

1. 未啟用雙重驗證以登入帳戶（社交媒體、銀行、遊戲等）

這使得駭客更容易破解您的密碼並在您未察覺的情況下完成登入。

{{% notice tip %}}

### 啟用雙重驗證 (2FA)
雙重驗證（通常簡稱為 2FA）是在登入時需要兩種形式（或因素）來證明您身份的做法。這十分有益，因為即使有人偷取或猜出您的密碼，他們仍無法登入您的帳戶。

雙重驗證需要在潛在的資料外洩事件發生**之前**啟用。大多數帳戶在您首次建立帳戶時便會要求您啟用，但如果您當時未設置，相關設定通常仍可在您的帳戶設定頁面找到。系統會要求您提供一個電話號碼或電子郵件地址，用來確認您的身份。一旦提供了第二個聯絡方式，您將收到一則確認訊息以確保您能收到這些驗證訊息。

啟用雙重驗證後，每次嘗試登入時，系統都會向您提供的電子郵件或電話號碼發送消息以確認登入者的身份。請確保在登入時手機在身邊！

{{% /notice %}}

2. 密碼重複使用（在多個網站使用相同的密碼）

一旦您的密碼被破解，駭客可能會輕易存取您的其他或所有網路帳戶。

{{% notice tip %}}

### 使用密碼管理工具

密碼管理工具是一個用於安全儲存密碼的地方，幫助您避免忘記密碼。使用它有幾個好處：

1. 您不需要記住多個使用者名稱和密碼，只需要記住解鎖密碼管理工具的密碼即可查看所有的密碼。
2. 您可以設定複雜且難以破解的密碼，並且不用擔心輸入它們的麻煩，因為您可以直接複製和貼上密碼。
3. 大多數密碼管理工具可以為您推薦強密碼，確保您不使用弱密碼。

市面上有許多密碼管理工具，甚至有些內建在網頁瀏覽器中！

{{% /notice %}}
