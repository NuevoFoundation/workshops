## Обзор

**C#: Негиздери** курсубузга кош келиңиз! Бул жерде сиз кызыктуу программаларды түзүү үчүн компьютерге нускамаларды берүүчү кеңири колдонулган, жогорку деңгээлдеги программалоо тилин үйрөнөсүз!

C# программалоо тили 2000-жылы Microsoft'tон Андерс Хейлсберг тарабынан иштелип чыккан. Бул курсту түзүү учурунда тилдин эң акыркы туруктуу версиясы 2022-жылы чыккан C# 11.0 болду.

{{% notice info %}}
Жогорку деңгээлдеги программалоо тили жогорку деңгээлде деп аталат, анткени ал компьютердин Борбордук процессордук бөлүмүндө (CPU) иштеген баштапкы коддон бир топ кадам алыс турат. Жогорку деңгээлдеги кодду адамдар колдонушу үчүн иштелип чыккан жана төмөнкү деңгээлдеги кодго караганда түшүнүү оңой. Жогорку деңгээлдеги код кийинчерээк белгилүү бир процессор тарабынан гана кабыл алынып, иштете турган төмөнкү деңгээлдеги тилге которулат. Код иштеши үчүн, жогорку деңгээлдеги тилде жазылган ар бир сап машина тилине которулуп, компьютер тарабынан ишке ашырылышы керек.
{{% /notice %}}

C# дүйнөдөгү эң популярдуу программалоо тилдеринин бири, Java, JavaScript, Python жана башкалар менен катар. C# Windows, Mac жана ар кандай Linux операциялык системаларында иштей алат.
