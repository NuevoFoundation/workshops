## Сиздин эң биринчи C# программаңыз

Биздин биринчи машыгуубузда консолго жазуу үчүн жөнөкөй C# тиркемесинин түзүлүшүнө көнүү максатында иштейбиз.

Келгиле, консолго `Hello World` басып чыккан C# программасын иштетип көрөлү. Консолго жазуу үчүн `Console.WriteLine ("Hello World");` коду колдонулат. Бул компьютерге скобканын ичиндеги текстти экранга жазууну буйруйт.

### Келгиле, код жазалы!

Жогорудагы .NET Fiddle рамкасында, биринчи саптын астына жаңы сап кошуп, "Салам *атыңыз*" деп басып чыгарууга аракет кылыңыз. Ал төмөндөгүдөй көрүнүшү мүмкүн.

<img src="../images/Step1.png" height="300" alt="Фиддлдин баштапкы экраны" />

Экрандын ылдыйкы бөлүгүндө сиздин чыгуу маанисиңизди көрүүгө болот. Супер, туурабы?  

Келгиле, башка нерселерди да басып чыгарып көрөлү. Кошумча код саптарын кошуңуз:
1. Сандарды консолго чыгаруу үчүн (мисалы, `Console.WriteLine(42);`). Мында кош тырмак ("") талап кылынбайт.
2. Тексттерди бириктирүү (мисалы, `Console.WriteLine("Hi " + "Sally");`).

{{% notice tip %}}
### Баардык жазылган саптар ар башка саптарга басылып чыкканын байкадыңызбы?

Кээде биз маалыматты бир сапта жазгыбыз келет. Бул үчүн `Console.WriteLine` ордуна `Console.Write` кодун колдонобуз.

Эки сап код кошуңуз:
```csharp
Console.Write("Мен жегенди жакшы көрөм ");
Console.Write("алма.");
```

Аны иштетип көрүңүз.
{{% /notice %}}

{{% notice warning %}}
### !! Маанилүү !! Кээде " " ичинде символдорду түзүп эле койсоңуз болбойт

Мурдагы символдорду `" "` ичинде чыгарып жатканда, кээде компьютер кайсы символдорду жазыш керектиги боюнча түшүнбөй калат.

Мисалы, компьютерге `"` жаз деп кантип буйрук беребиз?

Эгерде сиз `Console.WriteLine(""");`, деп жазсаңыз, ката ала аласыз, анткени компьютер текст кайсы жерден бүтөрүн түшүнө албай калат!

Ошондуктан, кээ бир символдорду чыгарып берүү үчүн, алдына `\` кошуу керек.

```csharp
Console.Writeline("\"");  // Бул " дегенди чыгаруу үчүн
```
{{% /notice %}}
