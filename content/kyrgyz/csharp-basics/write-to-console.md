## Сиздин биринчи C# программасы

Биздин биринчи көнүгүү — бул консолго жазган жөнөкөй C# колдонмосунун түзүлүшүнө көнүү.

Келгиле, консолго `Hello World` басып чыгарган C# программасын иштетип көрөлү. `Console.WriteLine ("Hello World");` деген код сап биздин бул ишти аткарат. Ал компьютерге () ичине жазылган нерсени экранга чыгар деп буйрук берет.

### Келгиле, код жазалы!

Жогорудагы .NET Fiddle бөлүгүнө биринчи саптын астына жаңы код сап кошуп, "Hello *атың*" текстинин басылып чыгышын камсыз кылыңыз. Ал төмөнкүдөй көрүнүшү керек.

<img src="../images/Step1.png" height="300" alt="Fiddle баракчасынын баштапкы көрүнүшү" />

Экрандын ылдый жагында жыйынтыгын көрөсүз. Сонун да, туурабы?  

Эми башка нерселерди басып көрөлү. Кошумча код саптарды кошуңуз:
1. Сандарды басып чыгаруу үчүн сандарды код ичинде көрсөтүңүз (мисалы, `Console.WriteLine(42);`). Кош жогорку тырмакча (" ") кереги жок.
2. Тексттерди бириктирүү (мисалы, `Console.WriteLine("Hi " + "Sally");`).

{{% notice tip %}}
### Байкадыңызбы, жазган бардык билдирүүлөрүңүз ар башка сапка басылды?

Кээде экранда бир сапка жазгыбыз келет. Бул үчүн `Console.WriteLine` ордуна `Console.Write` деген код колдонулат.

Эки сап кодду кошуңуз:
```csharp
Console.Write("I like to eat ");
Console.Write("apples.");
```

Жана аны тосо иштетип көрүңүз.
{{% /notice %}}

{{% notice warning %}}
### !! Маанилүү !! Кээде " " ичинде каармандарды жаза албайсыз

Басып чыгаруу билдирүүсүндө `" "` ичинде белгилүү каармандарды жазганда, компьютер кайсы каармандарды басуу керектигин түшүнбөй калуусу мүмкүн.

Мисалы, компьютерге `"` белгини басып чыгаруу үчүн кантип айтса болот?

Эгер сиз `Console.WriteLine(""");` деп жазсаңыз, ката пайда болот, анткени компьютер текст кайда бүтөрүн аныктай албайт!

Тескерисинче, белгилүү каармандарды басып чыгаруу үчүн аларды <b>качуу</b> керек, же болбосо алардын алдына `\` коюлат.

```csharp
Console.WriteLine("\"");  // бул " белгини басып чыгарат
```
{{% /notice %}}
