# Korean Snack Picker: final code

Reference solution for the [JS: Korean Snack Picker](../../../content/english/js-snack-picker/) workshop.

## Run it

1. Download or copy this folder to your computer.
2. Double-click `index.html`.
3. Click **Pick a snack!** and enjoy. 🍡

No installs, no build steps, just open the file in your browser.

## Files

| File         | What it does                                    |
|--------------|--------------------------------------------------|
| `index.html` | Page structure (button, card, headings)         |
| `styles.css` | Pastel theme, card styling, pop animation       |
| `script.js`  | Snack list, random picker, click handler        |
